# Fashion Editorial Landing Page

A visually styled fashion editorial landing page built using pure HTML and CSS.  
The project focuses on creative layout composition, modern typography, layered image overlays, and responsive hero section design inspired by luxury fashion magazines.

---
## 📸 Preview

![Project Preview](./images/fashion-editorial-banner.png)

---
## ✨ Features

- Editorial-inspired landing page UI
- Custom typography using `@font-face`
- Layered image composition
- Vertical outlined text effect
- Hover interactions and CTA button
- Responsive layout for mobile devices
- Pure HTML and CSS implementation

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Flexbox
- Media Queries
- Custom Fonts
